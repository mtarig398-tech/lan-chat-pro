# LAN Chat Pro Sprint 2.2.2

## التثبيت
1. شغّل `supabase/sprint-2.2.2.sql` بدور postgres.
2. استبدل `src/main.jsx`.
3. استبدل `src/styles.css`.
4. ضع ملف صوت قصير باسم `notification.mp3` داخل `public`.
5. اعمل Commit وانتظر Vercel.

## الميزات
- العداد داخل دائرة خضراء مثل واتساب.
- عداد إجمالي فوق الجرس.
- آخر رسالة ووقتها.
- إخفاء العداد عند فتح المحادثة.
- زر الرجوع يعمل داخل التطبيق أولًا.

## ملاحظة
إشعار النظام باسم المرسل والصوت أثناء إغلاق الموقع بالكامل يحتاج Service Worker/Web Push.
تعديل وحذف المجموعة يحتاجان واجهة Group Info منفصلة؛ ملف SQL هنا يجهز صلاحيات الحذف والتعديل.
